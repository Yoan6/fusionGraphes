Le site de recherche d'information sur les communes de France a pour objet de fournir des informations sur les communes de France, incluant des données démographiques, économiques, touristiques, et sur les élus municipaux.

2. Sources des données

Les données présentées sur ce site proviennent de différentes sources, notamment :
  • Wikipédia : Les contenus textuels et descriptifs des communes.
  • Répertoire des élus municipaux : Informations sur les élus locaux.
  • DATAtourisme : Données touristiques économiques et culturelles sur les communes de France.

Ces données sont récupérées automatiquement lors de la sélection d'une commune afin de garantir leur exactitude en temps réel. Cependant, le propriétaire du site ne peut être tenu responsable des erreurs ou omissions dans les informations fournies par ces sources externes.

3. Fonctionnalités du site

Le site permet 2 fonctionnalités principales :

  • la recherche d'information pour une commune de France donnée
  • L'exportation du site généré d'une commune de France précise

4. Légalité des données

Les données provenant des différents sites source sont open source mais doivent être référencées.
